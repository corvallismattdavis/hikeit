HikeIt

This will be a stand alone android application, we are starting from scratch and building it forwards to become a complete android application. Researching topics include: how to create and develop android applications, the linkage of a backend to a front end for data access, proper version control, and analysis of how other people have tried to solve this problem. For any application creativity is key to keep the user interested and engaged with design in mind to keep things easily accessible..

The HikieIt project is organized into the following directories:

  • features list -- what we would like to see in the app
  • story boards -- a few mock ups of what we're planning for the apps UI
  • testing -- testing information and ideas
  • documentation -- useful links, a directory for use to hold store information on our app
